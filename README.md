## Bilgisayarınızda Çalıştırın


Gerekli paketleri yükleyin

```bash
  npm install
```

Sunucuyu çalıştırın

```bash
  npm run start
```

Node.js BerberProject/berber/backend/
```bash
  node -v npm
```

BerberProject/berber/adminpanel/

```bash
  node -v npm
```
